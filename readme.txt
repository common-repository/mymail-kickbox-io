== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=MyMail+Kickbox.io+Integration) and an account at [Kickbox.io](https://kickbox.io)

Verifies your subscribers email addresses with kickbox.io

== Installation ==

1. Upload the entire `mymail-kickbox-io` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => Kickbox.io and enter your API Key
4. Enter your credentials
5. Send a testmail
